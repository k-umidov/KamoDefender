Три Критические Проблемы

Слабые Пароли

"123456", "password", "qwerty" — топ-3 самых популярных пароля в мире. Взломать их можно за секунды.

Фишинговые URL

Поддельные сайты копируют известные бренды. Пользователь вводит данные на "g00gle.com" вместо "google.com".

Вредоносные Файлы

"document.pdf.exe" — вирус маскируется под PDF. Один клик = заражение всей системы.

Как Работает Проверка Пароля

Password Security Analyzer использует научный подход к оценке надежности:

1. Система Баллов (0-100)

  • Длина ≥ 12 символов → +25 баллов
  • Заглавные буквы (A-Z) → +20 баллов
  • Строчные буквы (a-z) → +20 баллов
  • Цифры (0-9) → +20 баллов
  • Спецсимволы (!@#$%^&*) → +15 баллов

2. Расчет Энтропии (Криптографическая стойкость)

Энтропия = длина × log₂(размер_алфавита) Пример: "hello" (5 символов, a-z) → 23.5 бит СЛАБЫЙ "He11o!W0r1d" (11 символов, все типы) → 72 бит СИЛЬНЫЙ

Чем выше энтропия → тем дольше компьютер подбирает пароль перебором. Пароль с 72 битами энтропии = миллиарды лет на взлом!

URL Phishing Detector

Система анализирует ссылки по 6 признакам фишинга:

  • HTTP вместо HTTPS — отсутствие шифрования
  • Подозрительные слова — "secure", "verify", "login" в домене
  • IP-адрес вместо домена — http://192.168.1.1/paypal
  • Слишком много поддоменов — verify.account.secure.paypal.fake.com
  • Бесплатные домены — .tk, .ml, .ga (любимые у мошенников)
  • Typosquatting — g00gle.com вместо google.com

Пример фишинга:

http://secure-g00gle-login.tk/verify-account

HTTP • Подозрительный домен • Бесплатная зона .tk • Подделка "google"

File Security Scanner

Анализ файлов на 4 критерия опасности:

  • Опасные расширения — .exe, .bat, .scr, .vbs, .dll
  • Двойное расширение (маскировка!) — document.pdf.exe
  • Подозрительные имена — crack, keygen, virus, hack
  • Странный размер — program.exe размером 500 байт

Безопасно:

report.pdf photo.jpg document.docx

Опасно:

document.pdf.exe crack-adobe.bat virus.scr

Важно: Для полной антивирусной проверки рекомендуется использовать VirusTotal или профессиональные решения. KamoDefender — первая линия обороны!
